Introduction

This guide shares strategies and tactics for getting better results from GPTs. The methods described here can sometimes be deployed in <a href="#">code</a> combination for greater effect. We encourage experimentation to find the methods that work best for you. Some of the examples demonstrated here currently work only with our most capable model, gpt-4. If you don't yet have access to gpt-4 consider joining the waitlist. In general, if you find that a GPT model fails at a task and a more capable model is available, it's often worth trying again with the more capable model.
